```bash

| PHP 5.4+ & MariaDB 10.3 kurulumu           |
| PHP opcache, suhosin, imagick yapılandırması |
| LiteSpeed Optimizasyonu                    |
| MariaDB yükünün düşürülmesi                 |
| HTTP/2 Aktivasyonu                          |

| Uzak MySQL Engelleme/kapatma (port 3306)    |
